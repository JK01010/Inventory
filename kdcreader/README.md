# kdcreader

A KDCReader SW Flutter plugin project.

## Getting Started

This project is a KDCReader SW Flutter Plugin to interact with Koamtac products (KDC, KPOS) and Software decoder(KDC8).

To use this SDK, you should add SDK path into dependencies in pubspec.yaml as below.

dependencies:
...
  kdcreader:
    path: ../../kdcreader_sw/kdcreader
